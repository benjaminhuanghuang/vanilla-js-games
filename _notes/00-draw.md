Draw circle
```
  ctx.beginPath();
  // start from 0 to 2*PI
  ctx.arc(x, y, radius, 0, Math.PI*2);
  ctx.fillStyle = "#ffcd05";
  ctx.fill();
  
  ctx.strokeStyle = "#2e3548";
  ctx.stroke();
  
  ctx.closePath();
```

    ctx.closePath();
